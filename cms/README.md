# cms 管理系统功能

- 登录
- 统计
- 列表查询
- 发布文章
- todo 列表
- webrtc 连麦

# cms 管理系统技术栈

- react:前端逻辑
- react-router:前端路由
- hook: 前端状态
- antd: 前端 UI
- mobx: 存储状态
- craco: 打包管理
